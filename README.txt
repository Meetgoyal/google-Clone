# Google Search Page Clone

This project is a **Google Search Page Clone** built with responsive UI and interactive features that mimic Google's search experience.

---

## Features

- **Responsive UI:**  
  The layout adjusts smoothly across different screen sizes including mobile, tablet, and desktop.

- **Dark Mode / Light Mode Toggle:**  
  Easily switch between dark and light themes for comfortable viewing in any environment.
  option for same is in settings option on page.

- **Search Functionality:**  
  - Enter your search query in the search bar.  
  - On hitting enter or clicking the search button, the app navigates to a results page (stimulated) displaying the search query.
